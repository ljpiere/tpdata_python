## Resumen de Secciones y Contenido
1. **Pregunta Guía** – ¿Cómo trabajar como un profesional de datos, no solo como un estudiante?
2. **Exportación de formatos** – `.ipynb` (interactivo), `.py` (scripts), `.md` (documentación), `.html` (reportes), `.slides.html` (presentaciones), `.pdf`; cuándo usar cada uno
3. **Por qué `.ipynb` no es ideal** – input y output en el mismo archivo; JSON con basura; imposible entender cambios en Git; alternativas
4. **Quarto** – qué es; formato de presentación (revealjs); formato de blog (html); otros formatos; cómo crear un reporte desde un `.py`
5. **Celdas en `.py`** – `# %%` para crear celdas; enviar código a terminal interactiva; ventajas sobre Jupyter para producción; configurar "Python Run Selection/Line" como Ctrl+Enter
6. **Atajos de teclado** – cambiar shortcuts: `Ctrl+Enter` (run selection), `Ctrl+Shift+C` (comment), `Ctrl+Shift+A` (format); otros útiles: `Ctrl+Shift+F`, `Ctrl+Shift+H`, `Alt+Z`, `Ctrl+Shift+P`
7. **Extensiones avanzadas** – Git Graph, Git Lens, Git History, Project Manager, HTML Preview, Prettier; extensiones de tarea: Macros, Peacock, Markdown All in One, Better Comments, TODO Tree, Excel Viewer, Quarto
8. **Actividad práctica – Breakout Rooms** – students configure their VSCode with the recommended shortcuts and install 3 extensions; spinner to pick presenter
9. **Cierre**
