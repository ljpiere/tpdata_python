## Objetivos de Aprendizaje
- Usar **bucles `for`** para iterar sobre listas, rangos y DataFrames
- Usar **bucles `while`** para repetir código mientras se cumpla una condición
- Combinar bucles con **condicionales** para filtrar y transformar datos
- Aplicar técnicas básicas de **Data Preprocessing**: valores nulos, duplicados y formateo

## Resumen de Secciones y Contenido
1. **Pregunta Guía** – ¿Cómo procesar miles de filas de datos sin escribir código repetitivo?
2. **Bucles `for`** – sintaxis básica; `enumerate()`; `range()`; `zip()`; list comprehension
3. **Bucles `while`** – sintaxis; cuándo usar; `break` y `continue`; ejemplo: buscar valor en lista
4. **Bucles + condicionales** – filtrar ventas válidas; tabla de conversión Celsius→Fahrenheit
5. **Data Preprocessing** – dataset con problemas; `isnull().sum()`; `fillna()` (mediana, moda); `drop_duplicates()`; `str.title()`, `str.strip()`
6. **Breakout Rooms** – 3 exercises: clasificar temperaturas, inversión con while, limpiar dataset pingüinos
7. **Kahoot** – 6 preguntas
