## Objetivos de Aprendizaje
- Comprender la **estructura del sistema de archivos Unix/Linux**
- Navegar y operar el sistema con **comandos básicos de terminal**
- Entender las **variables de entorno** y el `PATH`
- Crear **alias y funciones** en `.bashrc` para productividad
- Usar **archivos `.env`** para almacenar secretos de forma segura
- Programar scripts con **crontab**

## Resumen de Secciones y Contenido
1. **Pregunta Guía** – ¿Por qué los Data Scientists necesitan saber usar la terminal?
2. **Unix/Linux** – por qué importa para DS (servidores, cloud, Docker); estructura de carpetas `/bin`, `/home`, etc.
3. **Comandos básicos** – navegación (`pwd`, `ls`, `cd`); archivos (`mkdir`, `cp`, `mv`, `rm`); Python (`python3`, `pip`, `venv`); permisos (`chmod`, `sudo`)
4. **Variables de entorno** – `env`, `echo $PATH`, `export`; archivos `.env` con `python-dotenv`
5. **Alias y funciones** – `.bashrc`; alias útiles para DS; función `new_ds_project()`
6. **Crontab** – formato; ejemplos: script diario a las 8am, cada hora
7. **Breakout Rooms** – configurar entorno: carpetas, git, venv, requirements.txt, alias
8. **Kahoot** – 5 preguntas
