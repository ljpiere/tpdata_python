# Students versions — notebooks generados

Estos notebooks fueron preparados como versiones para estudiantes: incluyen notas principales, espacios para apuntes, ejercicios abiertos, outputs limpios y secciones de respuesta/solución reemplazadas por espacios de trabajo cuando eran explícitas.

## Reglas aplicadas

- Se conservó la estructura general de cada clase.
- Se añadió una sección inicial de uso del notebook.
- Se añadieron notas de clase principales por notebook.
- Se limpiaron outputs y execution counts para ejecución desde cero.
- Se reemplazaron respuestas esperadas y soluciones explícitas por espacios para que el estudiante complete.
- En notebooks prácticos con código, varias celdas de solución fueron convertidas en celdas TODO conservando setup/imports/carga de datos.
